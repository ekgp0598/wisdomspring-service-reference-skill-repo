# HM-030 상단 메뉴

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 공통 화면 ID | HM-030 |
| 모바일 화면 ID | M-HM-030 |
| 웹 화면 ID | W-HM-030 |
| 화면명 | 상단 메뉴 |
| 화면 유형 | 메뉴 레이어 / 드로어 |

## 화면 개요

GNB 영역의 햄버거 메뉴 클릭 시 펼쳐지는 공통 메뉴 레이어다. 홈 이동, 테마 보기, 오늘의 질문, 마이 프로필, About, 공지사항, FAQ 등 주요 진입 링크를 제공한다.

## 플랫폼 정책

### PC 웹

- 현재 드롭다운 구조가 아니라 사이드 드로어로 노출한다.
- 메뉴가 길어지면 메뉴 레이어 자체에 스크롤이 발생한다.

### 모바일 앱 / 모바일 웹

- 햄버거 메뉴 진입 구조는 유지하되, 공통 메뉴 항목 정책은 동일하게 적용한다.
- `앱푸시설정`은 모바일 앱에서만 노출한다.
- PC 웹과 모바일 웹에는 `앱푸시설정`을 노출하지 않는다.

## 메뉴 구성

### 1. 홈

- 선택 시 `HM-001 메인(발견)`으로 이동한다.

### 2. 테마 보기

- 테마 리스트를 펼쳐서 바로 노출한다.
- 아코디언처럼 열고 닫는 방식이 아니다.
- 전체 테마를 한 번에 나열하며, 하단 메뉴는 아래로 밀린다.
- 메뉴 전체 길이가 길어지면 메뉴 레이어 내부에 스크롤이 생긴다.

### 3. 오늘의 질문

- 선택 시 `FD-001 질문피드`로 이동한다.

### 4. 마이 프로필

- 선택 시 `PR-001 프로필`로 이동한다.

### 5. About / 공지사항 / FAQ

- 기존 정책과 동일하다.
- About은 `CS-010 About(멤버십 소개)`로 이동한다.
- 공지사항은 `CS-020 공지사항`으로 이동한다.
- FAQ는 `CS-050 자주묻는질문(FAQ)`로 이동한다.

## 삭제 / 제외 정책

- 로그아웃 링크는 상단 메뉴에서 삭제된다.
- 로그아웃은 `PR-001 프로필 > 설정` 경로에서만 가능하다.

## 검증 포인트

- PC 웹에서는 메뉴가 드롭다운이 아니라 사이드 드로어로 떠야 한다.
- 테마 보기는 접힘 상태 없이 전체 리스트가 한 번에 펼쳐져야 한다.
- 테마 리스트가 길어질 때 하단 메뉴가 아래로 밀리고, 메뉴 레이어 자체에서 스크롤이 가능해야 한다.
- `앱푸시설정`은 모바일 앱에서만 보여야 하며, PC 웹과 모바일 웹에서는 보이면 안 된다.
- 로그아웃 링크는 메뉴에 노출되면 안 된다.
