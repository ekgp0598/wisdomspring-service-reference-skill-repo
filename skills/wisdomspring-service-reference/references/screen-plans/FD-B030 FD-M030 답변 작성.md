# FD-B030 / FD-M030 답변 작성

## 기본 정보

| 항목 | 내용 |
| --- | --- |
| 공통 화면명 | 답변 작성 / 답변 쓰기 |
| 모바일 화면 ID | M-FD-B030 |
| 웹 화면 ID | W-FD-M030 |
| 모바일 형태 | 바텀시트 |
| 웹 형태 | 모달 팝업 |

## 화면 개요

오늘의 질문에 대한 답변을 작성하는 오버레이 화면이다. 모바일은 현재 화면 위 바텀시트, PC는 팝업으로 제공하며 저장 완료 후 답변조회 흐름으로 연결한다.

## 상세 설명

### 1. 진입 경로

- 모바일 전용 인트로에서 `내 생각 쓰기` 클릭 시 진입한다.
- 홈(발견)에서 `작성하기` 아이콘 클릭 시 진입한다.
- 질문피드에서 `내 생각 쓰기` 또는 `작성하기` 아이콘 클릭 시 진입한다.

### 2. 저장 완료 후 동작

- 모바일은 저장 완료 후 AI 피드백보기 팝업 없이 답변조회 상세 화면으로 이동한다.
- 모바일 답변조회 상세의 상단 뒤로가기 선택 시 오늘의 질문 페이지로 이동한다.
- PC는 저장 완료 후 오늘의 질문 페이지로 이동하며 답변조회 팝업을 노출한다.
- PC 답변조회 팝업의 닫기 선택 시 팝업만 닫고 오늘의 질문 페이지를 유지한다.

### 3. 질문 영역

- 질문은 최대 2줄 높이로 고정 노출한다.
- 질문 글자 수 제한은 Admin에서 관리한다.

### 4. 답변 작성 영역

- 답변 작성 입력 영역을 제공한다.
- 줄바꿈은 허용한다.
- 입력 박스 높이를 초과하는 경우 내부 스크롤 처리한다.
- 글자 수 제한은 Admin에서 관리한다.

### 5. 옵션

#### AI 피드백 받기

- `AI 피드백 받기` 체크박스를 제공한다.
- 기본값은 체크 상태이다.
- 저장 직후에는 AI 피드백보기 팝업을 노출하지 않는다.
- 저장 완료 후 사용자는 답변조회 흐름으로 이동한다.

#### 나만보기

- `나만보기` 토글을 제공한다.
- 기본값은 체크 해제 상태이다.
- `나만보기` 설정 시 다른 사용자에게는 노출하지 않는다.
- 본인에게만 노출된다.
- 댓글 영역에는 `나만보기` 표시를 함께 노출한다.

### 6. 닫기 동작

- 작성 중 내용이 있는 상태에서 `X` 닫기 버튼 클릭 시 알럿을 노출한다.
- 알럿 문구는 아래와 같다.
- `작성 중인 내용이 저장되지 않을 수 있습니다. 창을 닫으시겠습니까?`

## 비고

- 답변 저장 후 이동 정책은 `latest-policy-updates.md`의 오늘의 질문과 답변 기준을 우선 적용한다.
